# CAF Sitio Web

Pensado para la gestión de prendas, muestras, clientes y usuarios.
Este proyecto está orientado tanto para celulares como tablets, y consume una API en .NET con base de datos SQL Server.

---

## 📋 Requisitos previos

Antes de comenzar, asegurate de tener instalado en tu entorno:

Node.js
 (versión LTS recomendada)

pnpm
 como gestor de paquetes


## 🚀 Cómo levantar el proyecto

1.  **Clonar el repositorio**

 ```
git clone https://github.com/YantornoJoel/CafFront.git
 ```

2. **Ingresar a la carpeta creada**

 ```
cd CafFront
 ```

3. **Instalar dependencias**

```
pnpm install
```

4. **Levantar el servidor de desarrollo**

```
pnpm run dev
```

https://nice-pond-00f989810.2.azurestaticapps.net/

### El proyecto se abrirá en tu navegador en http://localhost:5173 (o el puerto configurado).

---

## 📖 Descripción del proyecto

### CafFront es una aplicación frontend que permite:

🔑 Inicio de sesión y manejo de rutas privadas

👕 Creación, modificación y eliminación de prendas

🧵 Gestión de muestras

👥 Administración de clientes

👤 CRUD de usuarios
